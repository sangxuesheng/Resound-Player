## 说明

本文档用于记录当前项目中已统一的音乐详情页架构，包括：

- 每日推荐

- 普通歌单
- 统一播放策略
- 统一详情页模型

建议先阅读《统一架构说明》以了解整体设计；如涉及动效统一接入，请继续阅读《统一入场动画规范》。
