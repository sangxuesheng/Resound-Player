# 初始发布说明

## Release Title

`Resound Player - Stable Initial Release`

## Tag 建议

- `v0.1.0`
- 或按你的后续版本策略使用首个稳定标签

## Release Summary

Resound Player 首个稳定版本已完成归档。本次发布提供可运行的桌面音乐播放器基础工程，并完成了主应用代码、Electron 启动链路、页面模块结构与项目文档的系统整理，可作为后续持续迭代与版本发布的稳定基线。

## Highlights

- 完成桌面端基础工程搭建，支持 `Vue 3 + Vite + Electron` 运行方式
- 接通内嵌 API 启动链路，提升本地开发与桌面运行的一体化体验
- 建立统一歌单详情页模型，收敛多来源内容的展示逻辑
- 建立统一播放链路，支持按需解析歌曲播放地址
- 初步覆盖首页、搜索、歌单、专辑、歌手、排行榜、MV、播客、用户中心等核心页面
- 完成项目文档、架构说明与动画规范整理

## Included in this Release

### 应用能力

- 首页推荐内容展示
- 搜索与结果跳转
- 歌单详情、专辑详情、歌手详情、用户详情
- 排行榜与榜单详情
- MV 列表与弹层播放
- 播客列表与播客详情
- 用户中心、历史记录、设置页
- 底部播放器与展开播放器

### 工程能力

- Electron 主进程与预加载结构
- Vite 前端开发与构建配置
- 内嵌 API 自动拉起机制
- 动画规范检查脚本
- 项目文档入口与架构说明
