# 播放队列面板 `PlayQueuePanel.vue`

本文档说明播放队列面板的实现方式、接入方法和关键设计决策，方便后续维护和复用。

## 作用

从页面右侧滑出的播放队列面板，展示当前播放列表（`playerStore.playlist`），支持：

- 查看当前播放队列
- 点击曲目切换播放
- 音浪图标/播放图标悬浮交互（复用 `PlayPauseButton`）
- 从队列中移除单曲
- 一键清空队列
- 底部显示当前正在播放曲目
- 空态引导

## 组件结构

```
PlayQueuePanel.vue
├── Teleport to="body"
│   ├── Backdrop（遮罩层，点击关闭）
│   └── Slide Panel（面板主体）
│       └── AnimatedAppear（内部入场动画）
│           ├── Header（标题 + 数量 + 清空按钮）
│           ├── Song List（歌曲列表）
│           │   └── Song Item × n
│           │       ├── PlayPauseButton（序号/播放图标切换）
│           │       ├── Cover（封面图 / 占位符）
│           │       ├── Info（歌名 + 歌手）
│           │       └── Remove Button（移除按钮，hover 显示）
│           ├── Empty State（空态）
│           └── Footer（正在播放信息）
```

## 数据流

```
PlayerBar 按钮点击
  → uiStore.togglePlayQueue()
    → v-if="uiStore.showPlayQueue" 控制显隐
      → PlayQueuePanel 读取 playerStore.playlist 渲染列表
        → PlayPauseButton 监听 playerStore.currentSongId 和 isPlaying 切换图标
```

### 依赖的 Store

| Store | 字段 | 用途 |
|-------|------|------|
| `uiStore` | `showPlayQueue` | 面板显隐控制 |
| `uiStore` | `togglePlayQueue()` | 切换显隐 |
| `playerStore` | `playlist` | 播放队列数据 |
| `playerStore` | `currentIndex` | 当前播放索引 |
| `playerStore` | `currentSongId` | 当前歌曲 ID（PlayPauseButton 使用） |
| `playerStore` | `isPlaying` | 播放状态（PlayPauseButton 使用） |
| `playerStore` | `currentTrack` | 底部 footer 展示 |
| `playerStore` | `playByIndex()` | 点击曲目播放 |
| `playerStore` | `removeFromPlaylist()` | 移除单曲 |
| `playerStore` | `clearPlaylist()` | 清空队列 |

## 关键设计决策

### 定位

面板使用 `Teleport to="body"` + `position: fixed`，固定在页面右侧：

```css
right: calc(var(--layout-left, 8px));
top: var(--layout-top, 8px);
bottom: calc(var(--player-bar-height, 84px) + var(--layout-top, 8px));
```

- 使用 `--layout-top`、`--layout-left`、`--player-bar-height` 等布局 token 与项目整体布局保持一致
- 不依赖 sidebar 宽度，与侧栏位置解耦
- 宽度 330px，与主内容区保持间距

### 播放页背景融合（CSS 变量桥接）

当面板在 `PlayerExpanded`（播放页）中打开时，需要背景与播放页调色板保持一致。

实现方式：`PlayerExpanded.vue` 中通过 `watchEffect` 将 8 个 CSS 变量同步到 `document.documentElement`：

| 变量 | 值来源 | 用途 |
|------|--------|------|
| `--expanded-bg` | `bgStyle.background` | 面板整体背景色/渐变 |
| `--expanded-panel-bg` | `--panel-bg` | 面板实色基底 |
| `--expanded-panel-bg-soft` | `--panel-bg-soft` | 面板浅色变体 |
| `--expanded-accent` | `--accent` / `palette.c3` | 强调色 |
| `--expanded-line-muted` | `--line-muted` | 分割线、边框 |
| `--expanded-text-main` | `#ffffff` | 主文字色 |
| `--expanded-text-soft` | `rgba(255,255,255,0.55)` | 次要文字色 |
| `--expanded-text-sub` | `rgba(255,255,255,0.72)` | 副文字色 |

播放页关闭时自动清理所有 `--expanded-*` 变量，面板回退为默认主题色。

`PlayQueuePanel` 中所有颜色属性使用双回退链：`var(--expanded-xxx, var(--original-token))`。

### 磨砂半透明背景

面板背景使用三层叠加 + `backdrop-filter` 实现磨砂玻璃效果：

```css
background:
  radial-gradient(ellipse 70% 35% at 50% 0%, rgba(255,255,255,0.07) 0%, transparent 100%),
  radial-gradient(ellipse 60% 30% at 50% 100%, rgba(0,0,0,0.05) 0%, transparent 100%),
  color-mix(in srgb, var(--expanded-panel-bg, var(--bg-solid)) 80%, transparent);
backdrop-filter: blur(16px) saturate(140%);
```

- 径向渐变提供磨砂纹理（顶部柔光 + 底部阴影）
- 基底 80% 不透明，20% 透明，透过部分可见播放页背景
- `backdrop-filter` 对透过部分施加模糊 + 饱和度，形成磨砂玻璃质感

### z-index 层级

| 元素 | z-index |
|------|---------|
| 遮罩层 | 70 |
| 面板主体 | 80 |
| PlayerExpanded 容器 | 60 |

面板 z-index 高于 PlayerExpanded（60），确保在播放页打开时浮于上方。

### 动画

面板使用两层 `<Transition>`：

| Transition | 动画 | 说明 |
|------------|------|------|
| `play-queue-fade` | `opacity` 0.22s | 遮罩层淡入淡出 |
| `play-queue-slide` | `translateX(20px)` + `opacity` 0.28s | 面板从右侧滑入滑出 |

内部内容使用 `AnimatedAppear variant="content" rhythm="overlay"` 接入全局入场动画体系。

列表项通过 `an-enter-text` + `--i` / `--rhythm-offset` CSS 变量实现错峰入场。

### 序号悬浮播放交互

歌曲序号区域使用 `PlayPauseButton` 组件（`src/components/ui/PlayPauseButton.vue`），实现四种视觉状态切换：

| 条件 | 显示 |
|------|------|
| 非当前曲目 + 未 hover | 序号数字（灰色） |
| 非当前曲目 + hover | 播放三角图标（accent 色） |
| 当前曲目 + 未 hover | 三柱音浪动画（accent 色） |
| 当前曲目 + hover | 暂停图标 |

`PlayPauseButton` 通过 `bindRowHover()` 监听父级 `.song-item` 的 mouseenter/mouseleave，实现整行悬浮触发切换。

### 列表样式

- 网格列：`40px 40px 1fr auto`（PlayPauseButton / 封面 / 信息 / 移除按钮）
- 封面 40px，圆角 8px
- hover 时行背景使用 `color-mix(in srgb, var(--expanded-accent, var(--accent)) 8%, var(--expanded-panel-bg, var(--bg-solid)))`
- `.song-item--playing` 高亮复用 `theme.css` 的全局规则，box-shadow 使用 `var(--expanded-accent, var(--accent))`
- 移除按钮默认隐藏，hover 行时显示，hover 按钮时变红

### 弹窗规范合规

- 面板背景：80% 不透明 + `backdrop-filter: blur(16px)`（播放页下为磨砂半透明，普通页面下回退 `var(--bg-solid)`）
- 所有文字颜色使用双回退链：`var(--expanded-text-main, var(--text-main))` 等
- 列表行 hover 背景混合 `var(--expanded-panel-bg, var(--bg-solid))`，保持不透明
- 遮罩层：`rgba(0, 0, 0, 0.25)`（半透明，属于规范允许的例外）

### 无障碍

- 所有交互按钮有 `aria-label`
- 尊重 `prefers-reduced-motion`
- 键盘 focus 态通过 `focus-visible` 保证

## 接入方式

### 1. 状态注册（已接入）

在 `src/stores/ui.ts` 中已有：

```ts
showPlayQueue: false,
togglePlayQueue() {
  this.showPlayQueue = !this.showPlayQueue;
},
```

### 2. 按钮触发（已接入）

`src/components/PlayerBar.vue` 中播放列表按钮：

```html
<AnimatedAppear
  tag="button"
  variant="control"
  rhythm="actions"
  :index="6"
  class-name="icon"
  :class="{ active: uiStore.showPlayQueue }"
  data-tooltip="播放列表"
  aria-label="播放列表"
  @click="uiStore.togglePlayQueue()"
>
  <ListMusic :size="14" />
</AnimatedAppear>
```

### 3. 组件挂载（已接入）

`src/App.vue` 的 template 中，在 `<PlayerBar />` 之后渲染 `<PlayQueuePanel />`。

## 扩建指南

如需在面板内添加新功能（如拖拽排序、批量操作等）：

1. `playerStore` 已有 `moveTrack()` 方法可供拖拽排序使用
2. 面板内部列表样式已对齐 `detail-page.css` 的 `.song-item` 体系
3. 新增操作按钮优先接入全局按钮基线（`.button-surface`）
4. 新增交互入口可通过 `PlayPauseButton` 的事件机制扩展
