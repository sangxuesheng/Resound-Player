# 登录态鉴权：useAuthAction 复用说明

## 概述

`useAuthAction` 是一个 Vue 3 composable，用于统一处理需要登录态的操作（如收藏、收藏至歌单等）的鉴权逻辑。它将以下两种场景封装为一个函数：

1. **未登录** → 弹出登录弹窗
2. **搜索用户方式登录**（`loginMode === 'uid'`）→ 弹出 toast 提示需要切换登录方式

## 使用方式

```ts
import { useAuthAction } from '../composables/useAuthAction';

// 在 setup 中初始化
const { authToast, checkAuth } = useAuthAction(
  '操作失败的提示文案',   // 搜索用户登录时显示的 toast 文字
  'like',                // login intent: 'like' | 'playlist'
);

// 在操作函数开头调用
async function someAction() {
  if (!checkAuth()) return;  // 鉴权失败则提前返回
  // 鉴权通过，执行实际操作
}
```

### 返回值

| 返回值 | 类型 | 说明 |
|---|---|---|
| `authToast` | `Ref<string>` | toast 文案，非空时模板应显示 toast |
| `checkAuth` | `() => boolean` | 鉴权函数，返回 true 表示可继续操作 |

### 模板中的 toast

```html
<transition name="toast-fade">
  <div v-if="authToast" class="toast">{{ authToast }}</div>
</transition>
```

## 实现原理

```ts
export function useAuthAction(toastMsg: string, loginIntent: 'like' | 'playlist') {
  const authToast = ref('');

  function checkAuth(): boolean {
    if (!userStore.isLogin) {
      showLoginModal(loginIntent);   // 弹出登录弹窗
      return false;
    }
    if (userStore.loginMode !== 'cookie' && userStore.loginMode !== 'qr') {
      authToast.value = toastMsg;     // 显示 toast 提示
      setTimeout(() => { authToast.value = ''; }, 5000);
      return false;
    }
    return true;
  }

  return { authToast, checkAuth };
}
```
