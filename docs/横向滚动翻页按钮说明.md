# 横向滚动翻页按钮说明

## 组件

`src/components/ui/HorizontalScrollRail.vue`

## 功能

横向滚动容器左右两侧的 `‹` / `›` 翻页按钮，用于快捷滚动到下一组可见内容。

## 行为

### 滚动逻辑

点击按钮时计算滚动量：

```
delta = Math.max(minScroll, floor(container.clientWidth * scrollRatio))
```

- **`minScroll`**（默认 `240px`）：最小滚动距离
- **`scrollRatio`**（默认 `0.72`）：滚动量占容器可视宽度的比例
- 滚动方向：左按钮向左滚动 `−delta`，右按钮向右滚动 `+delta`
- 滚动方式：`scrollBy({ left: ±delta, behavior: 'smooth' })`

### 显示/隐藏

- 鼠标移入或焦点进入容器区域时显示按钮（`opacity: 1`、`pointer-events: auto`）
- 3 秒无交互后自动隐藏（由 `controlsTimeout` 控制，默认 `3000ms`）
- 移动端（`max-width: 767px`）默认隐藏（`hideControlsOnMobile: true`）

### CSS 变量

按钮尺寸和偏移量可通过 CSS 变量覆盖：

| 变量 | 默认值 | 说明 |
|---|---|---|
| `--horizontal-scroll-btn-size` | `37px` | 按钮宽高 |
| `--horizontal-scroll-btn-offset` | `-12px` | 按钮相对容器的水平偏移 |

其余按钮样式（背景、阴影、hover 态）复用 `.button-surface` 基线。

## Props

| Prop | 类型 | 默认值 | 说明 |
|---|---|---|---|
| `ariaLabel` | `string` | `'横向内容'` | 无障碍标签，用于拼接左右按钮的 `aria-label` |
| `contentClass` | `string \| array \| object` | `''` | 滚动内容区额外 class |
| `contentLayout` | `'grid' \| 'flex' \| 'block'` | `'grid'` | 内容区布局模式 |
| `minScroll` | `number` | `240` | 最小滚动距离（px） |
| `scrollRatio` | `number` | `0.72` | 滚动量比率 |
| `controlsTimeout` | `number` | `3000` | 按钮自动隐藏延迟（ms） |
| `hideControlsOnMobile` | `boolean` | `true` | 移动端是否隐藏按钮 |

## Emits

| Event | 参数 | 说明 |
|---|---|---|
| `rail-scroll` | `(event, element: HTMLElement)` | 内容区滚动时触发 |
| `rail-wheel` | `(event: WheelEvent, element: HTMLElement)` | 内容区滚轮时触发 |

## Expose

| 方法 | 说明 |
|---|---|
| `scrollToStart()` | 将滚动容器重置到起始位置 |
| `getRailElement()` | 返回滚动容器的 DOM 元素 |

## 使用示例

```html
<HorizontalScrollRail
  aria-label="热门歌手"
  content-layout="grid"
  @rail-scroll="onScroll"
>
  <ArtistCard v-for="artist in artists" :key="artist.id" :artist="artist" />
</HorizontalScrollRail>
```
